# DAILY-EXPENSE-CRUD-MANAGER

Description :-
Daily Expense CRUD Manager is a simple and efficient Python Project which helps individuals track there daily finance. 
The CRUD functionality i.e. Create, Read, Update and Delete, makes it easier to manage spending habits, maintain records and gain better financial awareness.

Features :-
My project contains 2 python files named :- 1. expense and 2. expense_tracker
1. expense contains constructor to initialise name, amount, category and subcategory variables
2. expense_tracker contains the operations that is to be done to organize and maintain expense records

Project contains following features :- 
1. Adds new expense records that includes name of expense, amount, category and and subcategory
   Category includes
     1. Needs
        1. Food
        2. Home
        3. Learn
     3. Wants
        1. Entertainment
        2. Shopping
        3. Others
3. View all recorded expense
4. Delete unwanted entries
5. Edit/Update any record

Tools Used :-
1. Programming Language - Python
2. Database - Microsoft Excel
